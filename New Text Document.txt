// A simple class with fields, constructor, and static method
public class MyClass {
    public int x;
    private string name;
    final float ratio = 0.5;

    private constructor(int x, string name) {
        super();
        this.x = x;
        this.name = name;
    }

    public void printInfo() {
        print(name + ": " + x);
    }

    public static void helper(MyClass obj) {
        print("Helper");
    }
}

// Namespace with enum
namespace someNameSpace {
    enum Colors {
        RED, GREEN, BLUE;
    }
}

// import for other file
import MyModule;

// Base class with abstract and virtual methods
private class Base {
    public abstract doAbstractThing(): void;

    public virtual doVirtualThing(): void {
        print("Base: doing virtual thing");
    }

    public final doFinalThing(): void {
        print("Base: doing final thing");
    }

    // Base constructors
    public constructor(string name) {
        print("Base constructor called with name: " + name);
    }

    public constructor() {
        // default constructor
    }
}

// Derived class overriding methods and using super calls
public class Derived extends Base {
    private string data;

    // Constructor calling super constructor
    public constructor(string name, string data) {
        super(name);
        this.data = data;
        print("Derived constructor called with data: " + data);
    }

    // Implement abstract method (override)
    public override doAbstractThing(): void {
        print("Derived: implemented abstract thing with data: " + this.data);
    }

    // Override virtual method, call base version via super
    public override doVirtualThing(): void {
        super.doVirtualThing(); // Call Base version
        print("Derived: extra virtual behavior");
    }

    // Static method (utility, not tied to instance)
    public static helperFunction(): void {
        print("Static helper called");
    }

    // Method without modifiers (METHOD_NONE)
    public regularMethod(): void {
        print("Just a normal method");
    }
}

1. Program Structure
A program is a sequence of declarations, definitions, and statements.

Top-level entities can be:

Class declarations

Enum declarations

import and namespace declarations

Function declarations

Variable declarations

2. Identifiers
Start with a letter or underscore, followed by letters, digits, or underscores.

Case-sensitive.

Cannot match keywords.

6. Classes
Syntax:

scss
Copy
Edit
[access] class ClassName [extends BaseClass] {
    // Fields
    [access] [final] Type name [= value];
    
    // Constructors
    [access] constructor(params) [: super(args)] { ... }
    
    // Methods
    [access] [modifier] name(params): ReturnType { ... }
}
Access Modifiers:

public – visible everywhere.

private – visible only inside class.


Method Modifiers:

abstract – no body; must be implemented by subclasses.

virtual – has default implementation; can be overridden.

override – replaces a base class method.

final – cannot be overridden.

static – belongs to the class, not an instance.

None – standard instance method.

7. Constructors

Overloading allowed (multiple constructors with different parameter lists)
